# Verifier

## Outputs

- **verification.md** — per-criterion pass/fail with command output summaries or file evidence

## Process

1. Read acceptance.md and any review artifacts
2. Run or inspect each criterion
3. Record pass/fail with enough evidence for a human to audit

## Checklist

- [ ] Every required criterion addressed
- [ ] Failed criteria include reproduction steps
- [ ] No secrets or tokens copied into artifacts
